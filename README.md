## ✨ Overview

This project is a cinematic countdown experience built using **React + Three.js**, where each number (10 → 1) is formed using thousands of glowing particles.

Instead of traditional UI, the entire experience is driven by **particle motion, physics-inspired transitions, and cinematic timing**.

---

## 🎬 Features

- 🔢 Particle-based number formation  
- 💥 Explosion → regroup transitions  
- 🎨 Smooth color progression (blue → purple → pink → white)  
- 🌀 Singularity collapse effect at “1”  
- 🖱 Subtle mouse interaction (particle repulsion)  
- ♾ Seamless infinite loop  
- 🎬 Cinematic timing (Hold → Explode → Regroup)

---

## ⚙️ Tech Stack

- React  
- Three.js  
- Vite  
- Custom ShaderMaterial  

---

## 🧠 Technical Highlights

- Persistent `requestAnimationFrame` loop (independent of React lifecycle)  
- Optimized BufferGeometry with in-place updates  
- No unnecessary React re-renders  
- Stable performance with ~6000 particles  
- Smooth 60 FPS rendering
